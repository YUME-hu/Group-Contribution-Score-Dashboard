# 小组贡献积分分红系统 (Group Contribution Score Dashboard)

一个面向大学生团队的轻量化贡献统计工具，用积分量化每个人的工作量，自动计算分红/奖励分配比例，彻底解决小组合作里「谁干多干少说不清、分配不公平」的老问题！

> 做竞赛、课程设计、毕设时，是不是总在纠结「奖金/学分该怎么分才公平？」
> 这个小工具帮你把贡献量化成积分，全程透明可追溯，再也不用为分配吵架！

## ✨ 核心功能
- 🧑‍🤝‍🧑 **灵活成员管理**：支持动态增删成员、自定义成员名称，适配2-15人不同规模的小组
- 📊 **智能积分计算**：按「任务难度等级+预估工时」自动计算贡献积分，支持手动微调补正
- 📈 **数据可视化**：积分占比环形图、贡献趋势折线图，团队贡献分布一目了然
- 📝 **操作日志追溯**：所有加分/修改操作全程记录，可回溯每一笔积分的来源
- 💾 **本地数据持久化**：数据存在浏览器本地存储，无需后端服务，隐私安全，刷新不丢失
- 📤 **数据导出备份**：支持导出CSV/JSON格式数据，方便留存分红记录、跨设备迁移

## 🎯 适用场景
- 大学生竞赛小组（计算机设计大赛/互联网+/软件杯/蓝桥杯等）贡献统计
- 课程设计/毕设小组工作量量化
- 实验室项目组任务分红
- 社团/学生组织干事绩效统计
